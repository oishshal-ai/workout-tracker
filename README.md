# Workout Tracker App

## Overview
A full-stack fitness tracking application built with:
- Frontend: React (JavaScript)
- Backend: Django
- Database: SQLite
- API: RESTful endpoints

This app allows users to log workouts and track progress.

---

## Current Functionality
- **Routing**
- **State Management**
- **CRUD Operations (local only)**

---

## Planned Functionality
- **Backend Integration**: Connect React to Django REST API.
- **Database Integration**: Replace local operations with Django + SQL backend
- **API Layer**: REST endpoints for workouts (create, read, update, delete).
- **Form Improvements**: Validation, error handling
---
