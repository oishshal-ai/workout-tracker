# User Workflows

This document describes the core user workflows in the Workout Tracker application.  
It complements the high-level overview in `README.md` by providing detailed steps for each action.

---

## Add Workout

1. Navigate to `/add`.
2. Fill out the workout form:
   - Title
   - Type (swim, water polo, strength, climb)
   - Additional fields (optional)
3. Click **Add Workout**.
4. The workout is saved and the user is redirected to the detail page (`/{type}/{id}`).

---

## Edit Workout (from detail page)

1. From a workout detail page, click **Edit**.
2. The edit form loads with existing workout data prefilled.
3. Update fields as needed.
4. Click **Save**.
5. The workout is updated and the user is redirected back to the detail page.

---

## Edit Workout (from dashboard)

1. Navigate to the **Dashboard**.
2. Locate the workout in the list.
3. Click the **Edit** button/icon
4. The edit form loads with existing workout data prefilled.
5. Update fields as needed.
6. Click **Save**.
7. The workout is updated and the user is redirected to the detail page.

---

## Delete Workout (from detail page)

1. From a workout detail page, click **Delete**.
2. Click the **Delete** button/icon

---

## Delete Workout (from dashboard)

1. Navigate to the **Dashboard**.
2. Locate the workout in the list.
3. Click the **Delete** button/icon
